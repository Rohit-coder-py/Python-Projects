<h2>About The Project</h2>

<p>

This project is a <mark>Machine Learning-based Employee Salary Prediction System</mark>
developed using <b>Python</b> and <b>Scikit-Learn</b>.

The project follows a complete ML workflow including <mark>Data Cleaning</mark>, <mark>Data Analysis</mark>, <mark>One-Hot Encoding</mark>, <mark>Train-Test Split</mark>, <mark>Feature Scaling</mark>, <mark>Model Training</mark>, and <mark>Model Evaluation</mark>.

Using employee-related information, the model predicts an estimated salary through <b>Linear Regression</b>.

This project helped me understand how Machine Learning models are built and deployed from raw data to final prediction.

</p>

</div>
