
<h1 align="center">🏦 Loan Approval Predictor</h1>

<p align="center">
A Machine Learning Classification Project that predicts whether a loan application is likely to be <b>Approved</b> or <b>Rejected</b>.
</p>

<hr>

<h2>📌 Project Overview</h2>

<p>
This project uses <b>Supervised Machine Learning</b> to analyze historical loan application data and predict loan approval status.
</p>

<p>
<b>Best Model:</b> Decision Tree Classifier<br>
<b>Accuracy Achieved:</b> 97.4%
</p>

<hr>
