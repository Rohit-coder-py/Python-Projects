# 🎓 Student Success Predictor

## Overview

A Machine Learning project that predicts student test scores based on study hours, attendance percentage, assignment completion, and gender using Linear Regression.

## Features

* Data Cleaning and Analysis
* Exploratory Data Analysis (EDA)
* Data Visualization
* Linear Regression Model
* Model Evaluation
* Student Score Prediction

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Use Cases

* Predict student academic performance
* Identify factors affecting test scores
* Support educational analytics
* Demonstrate an end-to-end Machine Learning workflow

## Model Evaluation

The model was evaluated using:

* MAE (Mean Absolute Error)
* MSE (Mean Squared Error)
* RMSE (Root Mean Squared Error)
* R² Score
