
#Restaurant Analytics System

This is my first data analysis project made using Python, NumPy, and Pandas.

In this project, users can explore restaurant data, search restaurants by city, filter by rating or budget, and view different types of restaurant analysis.

---

# Features

- View all restaurants
- Search restaurants by city
- Filter restaurants by rating
- Filter restaurants by budget
- View top-rated restaurants
- View cheapest restaurants
- City-wise analysis
- Cuisine analysis

---

# Technologies Used

- Python
- NumPy
- Pandas
- Jupyter Notebook

---

# Dataset

The dataset contains:

- Restaurant Name
- City
- Rating
- Price
- Cuisine

The dataset was created for practice and learning purposes.

git clone <your-github-link>
